# In-process channel adapter allowlist (sidecar-first policy).
#
# LibreFang is sidecar-first: new channels ship as out-of-process
# sidecar adapters (see docs / examples/sidecar-channel-python).
# This file grandfathers the in-process adapters that predate the
# policy. The pre-commit hook and `cargo xtask ci` reject any NEW
# file under crates/librefang-channels/src/{<name>.rs, <name>/mod.rs}
# that contains `impl ChannelAdapter` whose basename is NOT listed
# here.
#
# This list only ever SHRINKS: when an adapter is migrated to a
# sidecar and its module deleted, remove its name here so it can
# never be reintroduced in-process. Adding a name back requires an
# explicit maintainer decision (and an RFC) — it is not routine.
