# LibreFang - Agent Operating System

LibreFang 是一个用 Rust 编写的开源 Agent 操作系统，是 OpenFang 的社区维护分支。

<Note>
- **官网**: [https://librefang.ai/](https://librefang.ai/)
- **GitHub**: [https://github.com/librefang/librefang](https://github.com/librefang/librefang)
- **特性**: 15 个 Rust crates，241K+ 代码行，16 层安全防护，15 个 Hands
</Note>

## 核心概念

### Hands - 自主工作的 Agent

Hands 是 LibreFang 的核心创新——预构建的自主能力包，独立运行、按计划执行，无需人工干预。

| Hand | 功能 |
|------|------|
| **Analytics** | 数据收集、分析、可视化、仪表板和自动报告 |
| **API Tester** | API 端点发现、请求验证、负载测试、回归检测 |
| **Browser** | Web 自动化、填写表单、点击按钮、多步骤工作流 |
| **Clip** | 长视频转短视频、自动字幕和缩略图 |
| **Collector** | 情报收集、目标监控、变更检测、知识图谱构建 |
| **Creator** | AI 媒体工作室、图像/视频/音乐/语音生成 |
| **DevOps** | CI/CD 管理、基础设施监控、部署自动化、事件响应 |
| **Lead** | 潜在客户发现、信息丰富、评分排序、定时投递 |
| **LinkedIn** | LinkedIn 资料优化、内容创作、社交拓展 |
| **Predictor** | 信号收集、推理链构建、校准预测、准确度追踪 |
| **Reddit** | 子版块监控、内容发布、回复管理、互动追踪 |
| **Researcher** | 深度研究、交叉引用、事实核查、结构化报告 |
| **Strategist** | 市场调研、竞争分析、商业规划、战略建议 |
| **Trader** | 多信号分析、多空博弈推理、风险管理、投资组合分析 |
| **Twitter** | Twitter/X 内容创作、定时发布、互动管理、绩效追踪 |

## 安装

### Cargo 安装（推荐）

```bash
cargo install --git https://github.com/librefang/librefang librefang-cli
```

### 从源码构建

```bash
git clone https://github.com/librefang/librefang.git
cd librefang
cargo install --path crates/librefang-cli
```

<Note>
**提示**: 安装脚本和 Docker 启动时会自动初始化。如需手动配置，可运行 `librefang init` 进入交互式设置向导。
</Note>

### Docker

```bash
docker pull ghcr.io/librefang/librefang:latest

docker run -d \
  --name librefang \
  -p 4545:4545 \
  -e ANTHROPIC_API_KEY=$ANTHROPIC_API_KEY \
  -v librefang-data:/data \
  ghcr.io/librefang/librefang:latest
```

### 验证安装

```bash
librefang --version
```

## 快速开始

### 1. 初始化

```bash
librefang init
```

这会创建 `~/.librefang/` 目录和默认配置文件。

### 2. 配置 API Key

LibreFang 需要至少一个 LLM 提供商 API Key：

```bash
# Anthropic (Claude)
export ANTHROPIC_API_KEY=sk-ant-...

# 或 OpenAI
export OPENAI_API_KEY=sk-...

# 或 Groq (有免费额度)
export GROQ_API_KEY=gsk_...
```

### 3. 编辑配置

编辑 `~/.librefang/config.toml`：

```toml
[default_model]
provider = "groq"
model = "llama-3.3-70b-versatile"
api_key_env = "GROQ_API_KEY"

[network]
listen_addr = "127.0.0.1:4545"
```

### 4. 启动守护进程

```bash
librefang start
```

启动后访问 http://127.0.0.1:4545/ 查看仪表板。

### 5. 激活 Hand

```bash
# 激活 Researcher Hand
librefang hand activate researcher

# 查看状态
librefang hand status researcher
```

### 创建自定义 Agent

创建 `my-agent.toml`：

```toml
name = "my-assistant"
version = "0.1.0"
description = "A helpful assistant"
author = "you"
module = "builtin:chat"

[model]
provider = "groq"
model = "llama-3.3-70b-versatile"

[capabilities]
tools = ["file_read", "file_list", "web_fetch"]
memory_read = ["*"]
memory_write = ["self.*"]
```

然后生成：

```bash
librefang agent spawn my-agent.toml
```

### 内置工具

- `file_read` - 读取文件
- `file_list` - 列出目录
- `web_fetch` - 获取网页
- `web_search` - 搜索网页
- `memory_store` - 存储记忆
- `memory_recall` - 回忆记忆
- `shell` - 执行 shell 命令
- `browser` - 浏览器自动化

```bash

librefang agent list
```

## 16 层安全系统

LibreFang 采用深度防御策略，包含 16 个独立的安全层：

1. **WASM 双计量沙箱** - 燃料计量 + 时代中断
2. **Merkle 哈希链审计追踪** - 加密链接的完整操作链
3. **信息流污点追踪** - 敏感信息从源到汇的追踪
4. **Ed25519 签名 Agent 清单** - 加密签名的身份和能力集
5. **SSRF 保护** - 阻止私有 IP 和云元数据端点
6. **Secret 零化** - 自动清除内存中的 API 密钥
7. **OFP 双向认证** - HMAC-SHA256 验证
8. **能力门** - 基于角色的访问控制
9. **安全头** - CSP、X-Frame-Options、HSTS
10. **健康端点重写** - 公开/私有健康检查分离
11. **子进程沙箱** - 环境清理 + 进程树隔离
12. **提示注入扫描器** - 检测覆盖尝试
13. **循环守卫** - SHA256 循环检测
14. **会话修复** - 7 阶段消息历史验证
15. **路径遍历防护** - 规范化和符号链接转义防止
16. **GCRA 速率限制器** - 成本感知的令牌桶
