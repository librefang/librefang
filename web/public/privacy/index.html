Privacy Policy

Last updated: March 2026

Overview

LibreFang is an open-source project. We are committed to protecting your privacy. This policy explains what data we collect and how we use it.

Data We Collect

When you visit librefang.ai, we collect minimal anonymous analytics:

  • Page visits — anonymous page view counts via our self-hosted counter (no cookies, no personal data)
  • Google Analytics — standard web analytics (anonymized IP, no personal identification)

Data We Don't Collect

  • No personal information (name, email, etc.)
  • No cookies for tracking purposes
  • No data sharing with third parties for advertising
  • No user accounts or login data on this website

LibreFang Software

The LibreFang software itself runs entirely on your own infrastructure. It does not phone home or send telemetry data to us. All agent data, conversations, and configurations stay on your machine.

GitHub

Our source code is hosted on GitHub. When you interact with our GitHub repository (issues, PRs, discussions), GitHub's privacy policy applies.

Changes

We may update this policy from time to time. Changes will be reflected on this page with an updated date.

Contact

Questions about this policy? Open an issue on GitHub.
